# Vision Assistant Pro (Визуальный помощник Pro)

**Vision Assistant Pro** — это комплексное дополнение доступности для NVDA, интегрирующее модели искусственного интеллекта Google Gemini. Оно предоставляет набор инструментов, предназначенных для помощи незрячим и слабовидящим пользователям в повседневных цифровых задачах — от распознавания изображений до продвинутого перевода и обработки документов.

*Примечание: Этот проект был изначально выпущен для сообщества в честь **Международного дня людей с ограниченными возможностями** с целью использования искусственного интеллекта для достижения большей цифровой независимости.*

## Ключевые возможности

*   **Умный перевод:** Автоматически определяет языки и переводит текст. Включает специальный режим работы с буфером обмена для лучшей совместимости с веб-браузерами.
*   **Умный диктант:** Инструмент голосового ввода, который слушает речь, исправляет грамматику/пунктуацию и вводит текст прямо в активное приложение.
*   **Визуальный анализ:** Описывает объект под курсором навигатора или всё содержимое экрана. Поддерживает интерактивные уточняющие вопросы через диалоговое окно чата.
*   **Решение CAPTCHA:** Распознаёт изображения CAPTCHA с экрана и автоматически вводит результат.
*   **Анализ документов:** Позволяет пользователям выбирать документы (PDF, изображения, текст) и задавать вопросы об их содержимом.
*   **Обработка текста:** Утилита в меню для обобщения, объяснения или исправления грамматики выделенного текста.
*   **Расшифровка аудио:** Преобразует аудиофайлы (MP3, WAV, OGG) в текст.
*   **Автообновление:** Автоматически проверяет наличие новых версий и устанавливает их.

## Установка

1.  Скачайте последний файл `.nvda-addon` со страницы **[Последних релизов](https://github.com/mahmoodhozhabri/VisionAssistantPro/releases/latest)**.
2.  Откройте файл, чтобы запустить установщик дополнений NVDA.
3.  Подтвердите установку и перезапустите NVDA.

## Настройка

1.  Перейдите в **Меню NVDA > Параметры > Настройки > Vision Assistant Pro**.
2.  **Ключ API:** Введите ваш ключ Google Gemini API (можно получить в Google AI Studio).
3.  **Модель:** Выберите желаемую модель ИИ (например, `gemini-2.5-flash-lite`).
4.  **Языки:** Настройте исходный, целевой язык и язык ответов ИИ.

## Сочетания клавиш

Во избежание конфликтов с системными раскладками или раскладками ноутбуков, все сочетания используют **NVDA+Control+Shift**:

*   `NVDA+Ctrl+Shift+T`: Перевести текст/объект
*   `NVDA+Ctrl+Shift+Y`: Перевести из буфера обмена (рекомендуется для браузеров)
*   `NVDA+Ctrl+Shift+S`: Умный диктант (переключение)
*   `NVDA+Ctrl+Shift+V`: Описать объект навигатора
*   `NVDA+Ctrl+Shift+O`: Описать весь экран
*   `NVDA+Ctrl+Shift+C`: Решить CAPTCHA
*   `NVDA+Ctrl+Shift+R`: Меню обработки текста
*   `NVDA+Ctrl+Shift+D`: Анализ документа
*   `NVDA+Ctrl+Shift+F`: Распознавание текста из файла (OCR)
*   `NVDA+Ctrl+Shift+A`: Расшифровать аудиофайл
*   `NVDA+Ctrl+Shift+L`: Прочитать последний перевод
*   `NVDA+Ctrl+Shift+U`: Проверить обновления
