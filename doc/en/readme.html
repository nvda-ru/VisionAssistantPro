Vision Assistant Pro

Vision Assistant Pro is an advanced, multi-modal AI assistant for NVDA. It leverages Google's Gemini models to provide intelligent screen reading, translation, voice dictation, and document analysis capabilities.

Global Shortcuts

NVDA + Shift + T : Smart Translator
Translates the selected text or the object under the cursor. It includes "Smart Swap" to automatically switch languages if needed.
NVDA + Shift + S : Smart Dictation
Converts your speech to text using AI.
  • Press Once: Start recording (High Beep). Speak naturally.
  • Press Again: Stop recording (Low Beep). The AI transcribes your speech, fixes grammar, and automatically types it into the active text field.
NVDA + Shift + R : Text Refiner & Custom Commands
Opens a menu to perform actions like Summarizing, Fixing Grammar, or executing your own Custom Prompts (see below).
NVDA + Shift + 6 : CAPTCHA Solver
Captures the CAPTCHA image, solves it, and automatically types the code. It supports various number formats.
NVDA + Shift + V : Object Vision
Takes a screenshot of the navigator object and describes it. Opens a chat window for follow-up questions.
NVDA + Shift + O : Full Screen Vision
Scans the entire screen. You can ask the AI to describe the overall layout, read visible text, or explain what is currently displayed on your monitor.
NVDA + Shift + D : Document QA
Select a file (PDF, TIFF, TXT, MD, PY) to chat with. You can ask questions about its content.
NVDA + Shift + A : Audio Transcription
Select an audio file (MP3, WAV, OGG) to transcribe it to text.

Configuration

Go to NVDA Menu > Preferences > Settings > Vision Assistant Pro.

  • API Key: Required. Get your key from Google AI Studio.
  • Model: Choose gemini-2.5-flash-lite (Fastest) or others.
  • Languages: Set Source, Target, and AI Response languages.
  • Captcha Mode: Choose between Navigator Object (Default) or Full Screen capture.
  • Custom Prompts: Define your own commands here.

Custom Prompts & Variables

You can create custom commands in the Settings dialog using the format: Name:Prompt Text.
Separate multiple commands with a pipe symbol | or new lines.

Use these variables to create powerful workflows:

Variable Description Supported Formats
[selection] Inserts the currently selected text. Text
[clipboard] Inserts the clipboard content. Text
[screen_obj] Attaches a screenshot of the Navigator Object. Image (PNG)
[screen_full] Attaches a screenshot of the Full Screen. Image (PNG)
[file_ocr] Opens a dialog to select a file. The AI will extract text or analyze the visual content.
If no prompt is given, it defaults to "Extract text".		 Images (PNG, JPG, WEBP), PDF, TIFF (Multi-page supported)
[file_read] Opens a dialog to select a text document. The content is read and sent to AI. TXT, PY, MD, HTML, PDF, TIFF
[file_audio] Opens a dialog to select an audio file.
If no prompt is given, it defaults to "Transcribe this".		 MP3, WAV, OGG

Example Custom Prompts:

  • Quick OCR:
    My OCR:[file_ocr]
    (Opens a file picker, sends the image/PDF/TIFF, and asks AI to extract text.)
  • Translate Image:
    Translate Img:Extract text from this image and translate to Persian. [file_ocr]
  • Analyze Audio:
    Summarize Audio:Listen to this recording and summarize the main points. [file_audio]
Note: File uploads are limited to 15MB. Internet connection is required. Multi-page TIFFs are automatically converted to a series of images for processing.
