# Vision Assistant Pro

**Vision Assistant Pro** — це комплексний додаток для NVDA, що забезпечує доступність та інтегрує моделі штучного інтелекту Gemini від Google. Він надає набір інструментів, призначених для допомоги незрячим і слабозорим користувачам у виконанні щоденних цифрових завдань, від розпізнавання зображень до розширеного перекладу та обробки документів.

*Примітка: Цей проєкт був спочатку представлений спільноті на честь **Міжнародного дня людей з інвалідністю** з метою використання штучного інтелекту для забезпечення більшої цифрової незалежності.*

## Основні характеристики

* **Розумний переклад:** автоматично визначає мови та перекладає текст. Включає спеціальний режим буфера обміну для кращої сумісності з веб-браузерами.
* **Розумне диктування:** інструмент голосового введення, який слухає мовлення, виправляє граматику/пунктуацію та вводить текст безпосередньо в активну програму.
* **Візуальний аналіз:** описує об'єкт під курсором навігатора або весь вміст екрана. Підтримує інтерактивні додаткові запитання через діалог чату.
* **Розпізнавання CAPTCHA:** розпізнає зображення CAPTCHA з екрану та автоматично вводить результат.
* **Аналіз документів:** Дозволяє користувачам вибирати документи (PDF, зображення, текст) й запитувати про їхній вміст.
* **Уточнення тексту:** Меню утиліти для узагальнення, пояснення або виправлення граматики вибраного тексту.
* **Аудіотранскрипція:** перетворює аудіофайли (MP3, WAV, OGG) у текст.
* **Автоматичне оновлення:** автоматично перевіряє наявність нових версій і безперешкодно їх встановлює.

Примітка: Розмір завантаження файлів обмежений 15 МБ. Для всіх функцій штучного інтелекту потрібне активне з’єднання з інтернетом. Багатосторінкові файли TIFF обробляються автоматично.

## Встановлення

1.  Завантажте останню версію файлу `.nvda-addon` зі сторінки **[Останні випуски](https://github.com/mahmoodhozhabri/VisionAssistantPro/releases/latest)**.
2.  Відкрийте файл, щоб запустити інсталятор додатка NVDA.
3.  Підтвердьте інсталяцію та перезапустіть NVDA.

## Конфігурація

1.  Перейдіть до **Меню NVDA > Параметри > Налаштування > Vision Assistant Pro**.
2.  **Ключ API:** Введіть свій ключ API Google Gemini (його можна отримати в Google AI Studio).
3.  **Модель:** Виберіть бажану модель штучного інтелекту (наприклад, `gemini-2.5-flash-lite`).
4.  **Мови:** Налаштуйте початкову мову, мову для перекладу та мову відповіді штучного інтелекту.

## Гарячі клавіші

Щоб уникнути конфліктів із системними комбінаціями клавіш, усі комбінації використовують комбінацію **NVDA+Control+Shift**:

* `NVDA+Ctrl+Shift+T`: Перекласти текст/об’єкт навігатора
* `NVDA+Ctrl+Shift+Y`: Перекласти буфер обміну (рекомендовано для браузерів)
* `NVDA+Ctrl+Shift+S`: Розумне диктування (увімкнути/вимкнути)
* `NVDA+Ctrl+Shift+V`: опис об'єкта навігатора
* `NVDA+Ctrl+Shift+O`: Описати весь екран
* `NVDA+Ctrl+Shift+C`: розгадати CAPTCHA
* `NVDA+Ctrl+Shift+R`: меню уточнення тексту
* `NVDA+Ctrl+Shift+D`: Аналіз документа
* `NVDA+Ctrl+Shift+F`: оптичне розпізнавання файлу
* `NVDA+Ctrl+Shift+A`: транскрибувати аудіофайл
* `NVDA+Ctrl+Shift+L`: промовити останній переклад
* `NVDA+Ctrl+Shift+U`: перевірити наявність оновлень
